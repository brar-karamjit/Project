About Project

This project contains three apps, First is a candy Store link and it is deployed at https://guarded-plains-11100.herokuapp.com/ . It was created with django and deployed using herkoku. Candy Store is a dynamic website which uses python, bootstrap and sqlite database. On the homepage you have multiple options such as to buy individual candies, assortment or to even take the whole box. Each action has it's logic written in views.py and it accordingly update the database and dynamically updates all other fields such as the stock and images. It uses 'candy' model which has three attributes (name, quantity and image url) and the objects of candy class are stored in the database. Here is a video walkthrough of the project. Please feel free to visit by clicking the obove given link.

Second app (hosted on github) is made with GIPHY.com public api. it asks user for and input and appends it to the url, which is used to fetch data. This App uses Javacript to override the existing media library with the data received via fetch command (in JSON format).

Third Application (hosted on github) provides us with information regarding coronavirus cases in Canadian Provinces. User inputs Date and Province code and gets the related information. Such as daily case count, total cases upto that date etc. More data can be obtained from console. For its data source, it uses opencovid.ca
